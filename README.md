# TimeKeep Payroll v58 Modular

This modular GitHub Pages build was reconstructed from the latest v58 changes:

- Soft delete / hide employee
- Team Timesheets hides deleted employees
- Permanent Delete Data button
- Firebase Functions callable integration for `deleteAuthUser`

Deploy by uploading this folder contents to your GitHub repository root and enabling GitHub Pages from `main` / root.

Cloud Functions must be deployed separately with Firebase CLI.
